# 隐私政策

最后更新日期：2026年4月29日

## 概述

OpenPass 是一个开源的离线 TOTP（基于时间的一次性密码）密码管理器 Chrome 扩展。

我们非常重视您的隐私，并承诺保护您的个人数据安全。

## 数据收集与存储

### ✅ 我们不收集任何用户数据
- **无云端存储**：所有数据都只存储在您的本地浏览器中
- **无数据上传**：不会将任何数据发送到我们或第三方服务器
- **无遥测跟踪**：不包含任何用户行为分析或跟踪代码

### 本地数据存储
所有数据都使用 Chrome `chrome.storage.local` API 存储在您的设备本地：

| 数据类型 | 存储方式 | 说明 |
|---------|---------|------|
| TOTP 密钥 | 加密存储 | 使用主密码 AES-256 加密 |
| 备份文件 | 可选加密 | 可选择使用主密码或自定义密码加密 |
| 设置配置 | 明文存储 | 备份频率、目录选择等偏好设置 |

## 用户数据控制

您完全拥有并控制自己的数据：

- **导出数据**：您可以随时通过扩展界面导出所有密钥的备份文件
- **删除数据**：您可以随时删除单个密钥或清空所有数据
- **卸载扩展**：卸载扩展会删除所有本地存储的数据

## 权限说明

本扩展请求的权限仅用于核心功能：

- `storage`：存储密钥和设置
- `alarms`：自动备份定时任务
- `fileSystem`：自定义备份目录（可选）
