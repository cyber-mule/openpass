# 功能特性

## 密钥管理

### 添加密钥

支持多种方式添加 TOTP 密钥：

1. **手动输入** - 点击扩展图标右上角 **+** 按钮，输入密钥信息
2. **右键菜单** - 在网页中的二维码图片上右键，选择「识别并添加」
3. **导入备份** - 从备份文件批量导入密钥

**密钥信息：**

| 字段 | 说明 | 必填 |
|------|------|------|
| 密钥 | Base32 编码的 TOTP 密钥 | ✅ |
| 站点 | 目标网站域名 | ✅ |
| 名称 | 自定义显示名称 | ❌ |
| 位数 | 验证码长度（6/8位） | 默认 6 位 |

### 编辑与删除

- 将鼠标悬停在密钥卡片上，显示操作按钮
- 点击编辑图标修改密钥信息
- 点击删除图标移除密钥（需确认）

### 搜索过滤

在管理页面或弹窗中输入关键词，实时过滤匹配的密钥。

## 验证码生成

### TOTP 标准

- 完全兼容 [RFC 6238](https://tools.ietf.org/html/rfc6238) 标准
- 支持 6 位和 8 位验证码
- 30 秒更新周期
- 基于 HMAC-SHA1 算法

### 实时倒计时

- 验证码卡片显示剩余有效时间（圆环进度）
- 倒计时结束前 5 秒显示警告色
- 自动刷新验证码

### 一键复制

点击验证码卡片即可复制到剪贴板，复制成功后显示提示。

## 智能匹配

### 自动识别站点

扩展自动识别当前访问的网站：

1. 解析当前页面 URL，提取域名
2. 与已保存的密钥站点进行匹配
3. 扩展图标显示匹配数量（Badge）

### 匹配规则

- 支持主域名匹配（如 `github.com` 匹配 `gist.github.com`）
- 支持子域名匹配
- 不区分大小写

### 自动填充

在登录页面的验证码输入框旁，自动显示匹配的验证码候选列表：

- 显示验证码和倒计时
- 点击即可自动填充
- 支持多个匹配时切换

## 主密码保护

### 功能说明

主密码用于保护敏感操作，设置后：

- 打开管理页面需要验证主密码
- 导出备份需要验证主密码
- 会话超时后需要重新验证

### 安全机制

- 主密码不存储明文，仅存储 PBKDF2 派生的验证哈希
- 会话密钥存储在内存中，浏览器关闭后失效
- 会话超时时间：15 分钟

### 设置主密码

1. 打开管理页面 → 设置 → 主密码
2. 输入并确认主密码
3. 建议使用强密码（8 位以上，包含字母数字符号）

## 数据存储

### 存储位置

所有数据存储在 Chrome Storage API 中：

| 数据 | 存储方式 | 说明 |
|------|---------|------|
| 密钥 | 可选加密 | 启用主密码后加密存储 |
| 站点列表 | 明文 | 用于 Badge 显示 |
| 设置 | 明文 | 用户偏好设置 |
| 备份快照 | 可选加密 | 本地历史版本 |

### 数据安全

- 数据仅存储在浏览器本地
- 不会上传到任何服务器
- 不会通过浏览器同步（除非用户主动启用）

## 管理页面

点击扩展图标右上角的「管理」按钮，进入完整管理页面。

### 功能模块

| 模块 | 功能 |
|------|------|
| 密钥列表 | 查看、搜索、编辑、删除所有密钥 |
| 导入导出 | 备份恢复操作 |
| 设置 | 主密码、自动备份、备份加密等配置 |
| 关于 | 版本信息、项目链接 |
